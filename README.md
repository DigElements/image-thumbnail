# image-thumbnail

A Polymer Element showing a stylized image thumbnail with on-click behavior.

### Example
```html
    <image-thumbnail
        source="http://source"
        style-class="style-class"
        click-listener="[[listener]]">
    </image-thumbnail>
```

### Dependencies

Dependencies are installed using [Bower](http://bower.io/):

    npm install -g bower
    bower install
